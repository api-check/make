# Make (Integromat) Integration for ApiCheck

Connect ApiCheck to 1000+ apps via Make (formerly Integromat).

## Features

### Actions
- **Lookup Address** - Look up an address by postal code and house number (NL, LU)
- **Verify Email** - Check if an email is valid, disposable, or greylisted
- **Verify Phone** - Validate phone numbers and get formatted output
- **Global Search** - Search for addresses, streets, cities, or postal codes

## Installation

1. Go to [Make](https://www.make.com)
2. Create a new scenario
3. Click the + button to add a module
4. Search for "ApiCheck"
5. Connect your account with your API key from [app.apicheck.nl](https://app.apicheck.nl)

## Usage

1. Add the ApiCheck module to your scenario
2. Select an action (Lookup Address, Verify Email, Verify Phone, Global Search)
3. Map data from previous modules or enter values
4. Connect to other apps in your workflow

## Example Scenarios

- **Google Forms → ApiCheck → Google Sheets** - Validate addresses from form submissions
- **Shopify → ApiCheck → Shipping** - Validate shipping addresses
- **Mailchimp → ApiCheck → Filter** - Clean email lists before sending
